# Collapsible tabs

Collapsible tabs are used when a large amount of content needs to be gathered in one place and displayed as categories, for example.

We usually use them to make complex content more digestible, such as displaying multiple tables for example, in order to keep an equal area of visibility over the entire content. Closed, it also helps to get a clean user interface and allows the user to interact with it if needed.
