# Login card

Login card is displayed when user want to connect to his console management platform, preference center...

## Architecture

![Login page - Sign-in](https://zeroheight-uploads.s3.eu-west-1.amazonaws.com/4508b730f15ddb05efd54f)

Background elements and colors are available to download.

![Login modal architecture](https://zeroheight-uploads.s3.eu-west-1.amazonaws.com/765ef28f50947460a0cc7b)

Login modal is always centered horizontally and vertically inside the screen. Graphics elements of the background move regarding the size of the screen.

### Reset Password

![Reset-password-cards](https://zeroheight-uploads.s3.eu-west-1.amazonaws.com/6e7d94531e805a58352300)

Reset password allow the user to change his password when he forgot-it. User access to a page by clicking on the link inside reset password instructions email that he receive earlier. After entering a new password and confirming it, a success message appear to confirm the password has changed with success. User is invited to sign-in again with the new password.
