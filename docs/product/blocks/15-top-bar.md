# Top bar

**Top bar allow user to access quickly** to several elements: Console choices, Settings, Help Center and User profile.

- [Top bar](https://kleos.didomi.io/62b372835/p/226b0a-top-bar/b/32ba7d)

### Placement

**The top bar remains at the top of the screen.** When the user scrolls down the screen, the contents of the module area slide under the top bar.
