- [Interactive](https://kleos.didomi.io/62b372835/p/576200-stepper/b/342bd7)

## Interactive stepper

Interactive Stepper is used actually in the SSO settings process. For each step, the user must perform a validation action to move to the next step. When a step is validated, the user can return to it later.
