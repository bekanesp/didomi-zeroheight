## Hintbox Component

| Name | Description | Default | Control |
| --- | --- | --- | --- |
| Title | Defines the hintbox title (string) |  |  |
| Header icon | Defines the header (illustration) icon (string) |  |  |
| Header icon position | Defines the position of the header illustration (string) | right | right, left |
| One Line text | Sets the description next to the title (boolean) | false | True, False |
| Variant | Defines the hint variant (string) | neutral | neutral, success, failure, warning, info, white |
| Icon | The hintbox icon, shown if no illustration is set (string) |  | neutral, success, failure, warning, warning-blue |
| Content Alignment | The alignment of content (icon and title) with respect to the box (string) | center | center, start, end |
| Show close button | Defines if the hintbox is closable (boolean) | false | True, False |
| Size | Defines the hintbox size (string) | md | sm, md |
