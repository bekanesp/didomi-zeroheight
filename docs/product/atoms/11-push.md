# Push

Pushes are displayed generally in the header to highlight a specific content to highlight to the user. It could be: links to console documentation, user guides, right side panel etc...

[ **Cross Consol** ]

- [Drawer push](https://kleos.didomi.io/62b372835/p/99d4dd-push/b/7596f6)
- [Documentation push](https://kleos.didomi.io/62b372835/p/99d4dd-push/b/699e3d)

## Drawer push

Drawer pushes are small pushes used in the header to open the [drawer](https://kleos.didomi.io/62b372835/p/7493d1-drawer) (hidden right panel containing a short and precise aid).

### Variation

When the user's screen width is less than 1280px, the drawer push is reduced to avoid overlapping elements, we only keep the call to action "good to know!". This allows calls to action to be given priority.
